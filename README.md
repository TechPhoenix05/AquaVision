# MicroPi

This repository contains the MicroPi project — a multi-disciplinary project containing mechanical designs, electronics, firmware, software, and AI models.

Directory layout (top-level):

- `docs/` — documentation, images, diagrams, and progress notes
  - `images/`
  - `progress/`
  - `diagrams/`
- `3d_models/` — 3D models and exports
  - `v7_components_obj/`
  - `stl_exports/`
  - `assembly/`
- `electronics/` — electronics design, wiring, Pi-specific configs, firmware
  - `raspberry_pi/`
  - `wiring/`
  - `firmware/`
- `src/` — application code, AI models, and processing pipeline
  - `ai_model/`
  - `processing_pipeline/`
  - `app/`
- `tests/` — test inputs and suites
  - `mechanical/`
  - `optics/`
  - `software/`
- `results/` — sample outputs and logs
  - `sample_output/`
  - `inference_logs/`

Files like `.gitkeep` are present in empty directories so they are tracked by Git.
